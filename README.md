# padelero-landing

Landing page de [Padelero](https://padelero.app) — la plataforma de pádel en Argentina.

## Stack

- **Next.js 15** (App Router)
- **Tailwind CSS**
- **TypeScript**
- **lucide-react** (íconos)

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## Imágenes

Las imágenes están en `/public/landing/`. Para reemplazarlas, subí nuevas con los mismos nombres:

| Archivo | Sección |

| `hero-pampa.jpg` | Hero principal |
| `aerial-barrio.jpg` | Stats (fondo) |
| `lifestyle-mates.jpg` | Para jugadores |
| `club-interior.jpg` | Para clubes |
| `torneo-bracket.jpg` | Torneos |
| `rooftop-ba.jpg` | Padelero en Argentina |
| `court-night.jpg` | Testimonios |
| `asado-quincho.jpg` | CTA final |

## Deploy

Conectá el repo a [Vercel](https://vercel.com) y se deploya automáticamente en cada push a `main`.

```bash
npm run build   # build de producción
npm run start   # servidor de producción local
```
